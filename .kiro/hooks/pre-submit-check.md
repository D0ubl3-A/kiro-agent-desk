# Pre-Submit Check

Before submitting:

1. Confirm eligibility.
2. Confirm `.kiro/` folder is committed.
3. Confirm the demo opens from a clean checkout.
4. Confirm README explains how to run.
5. Confirm write-up is under 300 words.
6. Confirm "How we used Kiro" note reflects real Kiro use.
7. Confirm demo video or live link is public or accessible.
8. Confirm no secrets or private data are included.
